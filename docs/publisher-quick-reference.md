# 📸 Instagram Publisher - Quick Reference

## ⚡ Cheat Sheet

### 🎯 Fluxo de Publicação (30-90 segundos)

```
1. Imagens → 2. Caption → 3. Preview → 4. Publicar → 5. Media ID ✅
```

### 📐 Especificações

| Tipo | Imagens | Caption | Tempo |
|------|---------|---------|-------|
| **Post Único** | 1 | 2200 chars | ~30s |
| **Carrossel** | 2-10 | 2200 chars | ~60-90s |

### 🖼️ Imagens

- ✅ JPG, PNG
- 📏 1080x1080 recomendado
- 📦 Máx 10MB por imagem
- 🔗 URLs públicas ou drag & drop

### 📝 Caption Formula

```
🪝 GANCHO (125 chars antes do "...more")
📝 DESENVOLVIMENTO (2-3 parágrafos curtos)
💡 DICA BÔNUS (insight extra)
❓ CTA (pergunta ou ação)
📌 Hashtags (10-20 tags)
```

### #️⃣ Hashtags Strategy

```
3-5 Grandes (1M+):    #business #entrepreneur
5-10 Médias (100K-1M): #solopreneur #digitalnomad
3-5 Nichadas (<100K):  #lifestylebusiness #systemsbuilding
```

### 🎨 Templates Disponíveis

| Template | Uso | Emoji |
|----------|-----|-------|
| **Educativo** | Ensinar conceitos | 📚 |
| **Storytelling** | Compartilhar jornada | 📖 |
| **Promoção** | Lançar produto/serviço | 🎯 |
| **Lista/Dicas** | Conteúdo rápido | 📝 |

### 💾 Atalhos

| Ação | Como |
|------|------|
| **Salvar rascunho** | Botão 💾 Salvar |
| **Carregar rascunho** | Botão 📁 Rascunhos |
| **Usar template** | Botão 📄 Templates |
| **Limpar tudo** | Botão 🔄 Refresh |
| **Reordenar slides** | Setas ↑ ↓ |

### 📊 Engagement Score

| Score | Significado |
|-------|-------------|
| **80-100** | Excelente - Alta probabilidade |
| **60-79** | Bom - Ajustes menores |
| **40-59** | Médio - Melhorar caption |
| **<40** | Fraco - Revisar estratégia |

### ⚠️ Error Codes

| Erro | Causa | Solução |
|------|-------|---------|
| **401** | Token inválido | Renovar OAuth |
| **400** | Dados inválidos | Verificar URLs/caption |
| **500** | Erro servidor | Tentar novamente |
| **Timeout** | Demora >60s | Verificar URLs, retry |

### 🚀 Checklist Pré-Publicação

- [ ] 1-10 imagens adicionadas
- [ ] Preview visualizado
- [ ] Caption escrita (<2200 chars)
- [ ] Hashtags incluídas (10-20)
- [ ] CTA presente
- [ ] Primeira linha é gancho forte?
- [ ] Emojis estratégicos (3-7)?

### 💡 Golden Rules

1. ✅ Carrosséis > Posts únicos
2. ✅ Primeira linha = tudo
3. ✅ 10-20 hashtags = sweet spot
4. ✅ CTA claro = +30% engajamento
5. ✅ 3-7 emojis = legibilidade
6. ✅ Parágrafos curtos = melhor leitura
7. ✅ Gancho visual no slide 1
8. ✅ CTA forte no último slide
