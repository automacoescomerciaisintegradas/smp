# Social Flow Design System v1.0.0

Este documento define os tokens visuais e padrões de interface para o projeto Social Flow.

## Palette
- **Primary:** `#E54D42` (Cinzento-avermelhado vibrante)
- **Background:** `#F9FAFB` (Default) | `#FFFFFF` (Paper) | `#F3F4F6` (Sidebar)
- **Text:** `#111827` (Primary) | `#6B7280` (Secondary) | `#9CA3AF` (Disabled)
- **Status:** `#10B981` (Success) | `#FEE2E2` (Error BG) | `#991B1B` (Error Text)
- **Border:** `#E5E7EB`

## Typography
- **Sans:** `Inter, system-ui, sans-serif`
- **Mono:** `JetBrains Mono, Fira Code, monospace`
- **H1:** `24px / 700`
- **Body:** `14px / 400`
- **Label:** `12px / 500`

## Components
- **Sidebar:** Width `240px`, BG `#F3F4F6`, Active Item BG `#FEE2E2`, Active Item Text `#E54D42`
- **Card:** Border Radius `8px`, Border `1px solid #E5E7EB`, Shadow `0 1px 3px 0 rgba(0, 0, 0, 0.1)`
- **Button:** Border Radius `6px`, Padding `8px 16px`, Font Size `14px`
